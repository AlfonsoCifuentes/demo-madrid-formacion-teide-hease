# demo-madrid-formacion-teide-hease

Demo conceptual no oficial para Grupo TEIDE-HEASE.

Sector: centros de formacion profesional
Ciudad: Madrid
Fuente: http://www.teideformacion.es/
Contacto publico: asesor.fp@teideformacion.com
Precio propuesto: 1.480 EUR

Punto debil trabajado:
La oferta es amplia, pero la navegacion tiene demasiados niveles antes de llevar al alumno o familia al programa adecuado.

Stack: HTML estatico, CSS embebido y vercel.json para despliegue en Vercel.
